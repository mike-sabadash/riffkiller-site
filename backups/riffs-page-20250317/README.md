# Backup: страница риффов (до редизайна 2025-03-17)

Чтобы откатить визуал и разметку главной с риффами:

1. Скопируйте `app.html` и `js/main.js` из этой папки в корень проекта (замените файлы).
2. В `app.html` удалите подключение `css/riffs-hub.css`, класс `riffs-hub` на `<body>`, блок `.riffs-hero`, обёртку `.riffs-panel` и замените шапку секции риффов на старый вариант с `.riffs-header` (см. файл `app.html` в этой папке).

Файлы:

- `app.html` — разметка страницы
- `main.js` — карточки и логика (на момент бэкапа)

Общие стили остаются в `css/components.css` (не копировались целиком — при необходимости откатывайте точечно по git).
